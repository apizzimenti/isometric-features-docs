Example
// example parameters:

var initial = {
    isoZ: 100
};

var tween = {
    properties: {
        isoZ: 24
    },
    duration: 2000,
    easing: Phaser.Easing.Bounce
}

var player = new Player (game, row, col, keys, group, map, null, false); // player is not set to be immediately created
player.addIntro(initial, tween);
player.create();

visionRadius()

Tracks the 3x3 vision radius of the Player, and changes tiles within that radius to visible. Is turned off if Map.fog === false.
