Globals

new Globals()

A set of variables that have to be used in disparate locations. All methods are static.
Properties:
Name Type Description
anchor Array.<number> Globalized anchor for all sprites.
mapTileKey Array.<string> Will contain keys for tile sprites.
tween array Default tween settings.
paramNotExist function Global testing method for parameters.
